用SED删除程序代码里的行号以及空格

Mar 31st, 2011 | Comments

有时候在网上搜索到的程序代码复制下来后会有行号，在代码行数较少的情况下可以手动删除，遇到上百行代码的时候要是手动删除就是对体力加毅力的考验，而且不符合电工风格，于是想到使用正则表达式来实现自动删除。 分别使用VIM和SED实现删除代码里的行号以及空格，最近才开始学正则表达式，语法参数还不是很熟悉，反复调试了几次后才成功的。

  1. 使用VIM

vim finlname,在command模式, :%s/[0-9][ ]*//

  1. 使用SED

sed -r ‘s/\s*[0-9]+[ ]+//’ filename >filename1

sed加-i参数可以直接修改文件，-i 后面指定扩展名则会生成备份文件，就可以不用管道命令。

在ubuntu10.10下安装tpfan

Feb 14th, 2011 | Comments

tpfan的最后一个版本0.96,只适用于ubuntu9.04，本文是基于T61而写。

1.下载tpfan的deb文件，http://ppa.launchpad.net/tp-fan/ppa/ubuntu/pool/main/t/tpfan-admin/tpfan-admin_0.96-ubuntu1_all.deb

2.解压后修改debian文件夹中的control文件，将“python-gnome2-desktop”替换为“python-gnome2-desktop-dev”

3.重写打包

dpkg-deb -D --build tpfan-admin_0.96-ubuntu1_all

4.安转

5.修改/usr/lib/python2.6/dist-packages/tpfand/settings.py7. 替换行 170-174

self.product_id = None
self.product_name = None
self.product_pretty_vendor = None
self.product_pretty_name = None
self.product_pretty_id = None

替换为

self.product_id = ""
self.product_name = ""
self.product_pretty_vendor = ""
self.product_pretty_name = ""
self.product_pretty_id = ""
sudo /etc/init.d/tpfand restart

Ubuntu10.10下实现指纹登录

Dec 13th, 2010 | Comments

1、安装thinkfinger软件

ocean@ocean-laptop:~$ sudo apt-get install thinkfinger-tools libpam-thinkfinger

2、采集指纹并校验：

ocean@ocean-laptop:~$ sudo tf-tool –acquire //采集指纹,连续扫描三次指纹
ocean@ocean-laptop:~$ sudo tf-tool –verify //校验指纹

第二步需要超级用户权限,否则不能驱动指纹传感器,,如果提示Initializing…USB device not found,建议重启一下系统,然后再进行这一步操作,由于博客主题代码问题”- -acquire”会显示为”–acquire”,操作的时候请注意

3、启用指纹功能

ocean@ocean-laptop:~$ sudo /usr/lib/pam-thinkfinger/pam-thinkfinger-enable

4、在debian最新的thinkfinger包中已经放弃原用的 –add-user参数,而是在第二次运行程序时仍然使用–acquire参数,自动生成后缀为bir的指纹文件存放在家目录下. 如果希望在su,sudo时也使用指纹识别(gksu,gksudo包括emacs的tramp也已经可以使用了)就以root用户身份登录后生成指纹文 件.然后重启,登录时先输入用户名,然后原来password处应该已经变成password or swipe finger了.之后就可以用刷指纹代替输入密码了.

除了百度MP3,你还有更好的选择-谷歌音乐

Dec 8th, 2010 | Comments

Google音乐是谷歌被迫适应中国国情而推出的一种新的音乐服务运作模式的实验，用户无需为正版音乐付费。谷歌通过在线音乐广告分成的模式让找到了一个提升用户体验、尊重歌手创作、尊重版权之间的平衡。

Google音乐与巨鲸音乐网（www.Top100.cn）合作，所有音乐文件均经过唱片公司正版授权，并提供免费高品质MP3文件下载，每个MP3文件的音质均在192Kbps以上。谷歌官方指其音乐库数量已达数十万首，并且在不断增加中.

题外话:如果你在使用Linux下的Rhythmbox,这里提供一个网友写的谷歌音乐插件,该插件可以在Rhythmbox里播放谷歌音乐,甚至支持拖拽歌曲到本地目录的下载方式.

下载googlemusic 解压在 /usr/lib/rhythmbox/plugins/googlemusic 目录下,或 ~/.local/share/rhythmbox/plugins/googlemusic 目录下.
